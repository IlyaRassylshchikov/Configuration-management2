# Инструмент визуализации графа зависимостей

Инструмент для визуализации графа зависимостей npm пакетов с построением полного графа транзитивных зависимостей и определением порядка загрузки.

## Этапы реализации

###  Этап 1: Минимальный прототип с конфигурацией
###  Этап 2: Сбор данных о зависимостях
###  Этап 3: Построение графа зависимостей
###  Этап 4: Порядок загрузки зависимостей

## Возможности

-  Анализ npm пакетов через npm registry API
-  Построение полного графа транзитивных зависимостей (BFS с рекурсией)
-  Обнаружение и обработка циклических зависимостей
-  **Определение порядка загрузки зависимостей (топологическая сортировка)**
-  Фильтрация (исключение) пакетов по подстроке
-  Настройка максимальной глубины анализа
-  Режим тестирования с локальными данными
-  Визуализация дерева зависимостей
-  Сравнение результатов с реальным менеджером пакетов
-  Обработка ошибок и предупреждений
